Generic property loader by Raphaël Mechali

The Generic property loader is a tiny project that aims at providing a flexible and easy to use API for loading property in JAVA applications/components. Generic, in the project name, means that the project has also for goal to type the data retrieved from property files, whatever that data type could be.
